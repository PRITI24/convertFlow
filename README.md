## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## How to Use

1. Open the app in your browser (usually at `http://localhost:5173` after running the dev server).
2. Select the conversion mode: PDF to Word or Word to PDF.
3. Click "Choose File" to upload your PDF or DOCX file.
4. Click "Convert" to start the conversion process.
5. Wait for the processing to complete.
6. Download the converted file using the download button.
